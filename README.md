# GrasshopperCode
🦗 Grasshopper plugin code

## Documentation
The website is live at https://ifchoppershell.github.io/grasshopper-documentation/

## Plugin
You can download the compiled version of the plugin on food4rhino at https://www.food4rhino.com/en/app/ifchoppershell or from Rhino Package Manager.

## Versions
**main** branch contains the code currently under developement, so it may be unstable. Use only if you are contributing to the code.

**v0.1.2** is the currently stable version, published to food4rhino and package manager.
